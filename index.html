About the mod

Doki Doki: The Angel Returns is an unofficial mod for the game Doki Doki Literature Club! that takes place after the events of the game (see IP Guidelines for details). This mod is the answer to a simple question: What if "Alice Angel" came to the Doki Doki universe?
